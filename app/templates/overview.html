BitBot

OVERVIEW

BitBot is an interactive tool used to encourage users to learn about trading crypto currencies and other stocks. It has built-in algorithms to show users different ways of analyzing stocks, and give them examples of when it would be good to buy in or sell. These examples will be performed on real-time data, giving users a much more valuable experience as we can show them extremely recent examples on real data. BitBotâ€™s ability to compare multiple algorithms and show that data to the user is important for learning as from there the user can extrapolate any data they might deem relevant or necessary and even save the plot to examine later. We provide the algorithms, and from this the user can learn and make more-educated decisions. It is our hope that one could utilize this tool to learn about different trading algorithms and become more successful in the cryptocurrency/stock industry.
